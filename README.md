# Jitsi-Games
An app that allows you to play html games with your friends! \
Built with:\
-Vite\
-Javascript\
-Express\
-Redis\
-Websockets\
-WebRTC(Peer.js) \

## Installation
```sh
git clone
```
client 
```sh
cd client/ npm install
npm run dev
```
server
```sh
cd server/ npm install
npm run start
```
redis
```sh
Install redis to your computer if you haven't already and redis-cli
sudo systemctl start redis
```
